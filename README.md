# CIQ-Use-case

Standalone Conscience IQ Use Cases page for Wix Studio embedding.

## Files

- `index.html` redirects visitors to the use cases page for GitHub Pages and simple static hosting.
- `use-cases.html` contains the page markup.
- `css/use-cases.css` contains the page styles.
- `js/use-cases.js` contains progressive navigation behavior.

## Local Preview

Open `index.html` or `use-cases.html` directly in a browser.
